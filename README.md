# Tesla and GameStop Stock Analysis

## Project Overview
This project analyzes the historical stock performance of Tesla and GameStop using Python. It includes stock price extraction, data cleaning, visualization, and business insights based on market trends.

## Objectives
- Extract Tesla stock data using yfinance
- Extract GameStop stock data using yfinance
- Clean and preprocess the datasets
- Visualize historical closing prices
- Compare long-term growth and volatility
- Generate business recommendations

## Tools and Libraries
- Python
- Pandas
- yfinance
- matplotlib
- Jupyter Notebook

## Files Included
- `tesla_stock_analysis.ipynb`
- `gamestop_stock_analysis.ipynb`
- `stock_analysis.py`
- `presentation.pdf`

## Key Insights
- Tesla shows strong long-term growth
- GameStop has very high volatility
- Visualization highlights different market behaviors
- Data-driven insights support better investment decisions

## Conclusion
This project demonstrates practical financial data analysis using Python and real stock market datasets. It helps compare stable growth stocks with highly volatile meme-stock behavior.
